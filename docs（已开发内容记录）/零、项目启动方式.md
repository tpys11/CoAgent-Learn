## 安装与启动——从零开始

1. 安装 Git：https://git-scm.com/downloads/win

2. 安装 Docker Desktop：https://www.docker.com/products/docker-desktop/

3. `git clone https://github.com/tpys11/CoAgent-Learn.git`

4. `cd CoAgent-Learn`

5. `docker compose -f deploy/docker-compose.yml up -d`

6. 浏览器打开 `http://localhost:5173`

7. 首次打开弹窗配置 DeepSeek API Key，保存即可使用

## 原理

CoAgent-Learn 由 4 个 Docker 容器组成，`docker compose up -d` 一键全启：

| 服务       |  端口  | 作用                               |
| -------- | :--: | -------------------------------- |
| frontend | 5173 | React 19 前端（Vite 代理 /api → 8000） |
| backend  | 8000 | FastAPI + LangGraph 多智能体协同       |
| redis    | 6379 | 缓存与任务队列                          |
| chroma   | 8001 | 向量数据库（RAG 知识检索）                  |

用户只需访问 5173 一个端口。
