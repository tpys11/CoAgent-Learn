# 二、界面搭建

把概念模型的全部内容放在界面上，便于后续查看。

## 技术栈

React 19 + TypeScript + Vite 6 + Tailwind CSS 3 + @xyflow/react 12

## 功能总览

### 1. 三栏布局

四周 8px 呼吸间距。左侧可拖拽拉伸（180-400px）/ 中间弹性 / 右侧可拖拽拉伸（180-400px）。左右栏均可折叠，折叠按钮在外侧。

### 2. 左侧栏 — 项目 + Agent

**上半部分：项目与对话管理（仿 workbuddy）**
- 顶部品牌区（Bot 图标 + 名称 + 新建按钮）
- 每个项目可展开/折叠，显示下属对话窗口列表
- 对话窗口支持：新建、重命名、归档
- 新建项目后自动创建首个对话窗口并弹出知识诊断

**下半部分：Agent 列表**（8 个 Agent）
- 🎯调度 / 🧠记忆管理 / 🔍学情诊断 / ⚖️审核裁判 / 📥输入信息处理 / 📤信息整理与生成 / 🔎搜索 / 📚知识库管理
- 点击 Agent 弹出配置模态框（模式/提示词/Skill/恢复默认）
- Agent 列表限高 50%，超出滚动

### 3. Agent 配置模态框

- 模式选择、全局性提示词编辑、Skill 编辑（部分可编辑）、恢复默认按钮

### 4. 中间 — 对话区

- **反馈横条**：时间范围下拉 + 专注时长 + Token 用量 + 实时时钟，可折叠（▲/▼）
- **AgentFlow 画布**：发送消息后出现，仅展示图标节点和黑色箭头连线，Agent 执行时节点放大高亮
- **思考过程**：灰色气泡实时流式展示 Agent 思考内容，完成后自动折叠为「▸ 思考过程」，可展开查看
- **消息气泡**：用户浅橙右对齐，助手透明左对齐，助手消息内嵌可折叠的思考过程
- **控制栏**：记忆/知识库/检索与搜索/输入优化/输出形式/输出内容 — 均弹出上拉面板，每选项有说明
- **输入框**：textarea + 橙色发送按钮，Enter 发送 Shift+Enter 换行

### 5. 右侧面板

- 知识图谱占位（虚线边框卡片）
- 追问对话区：展开后逐个弹出追问气泡 + 底部输入框

### 6. 知识诊断弹窗

三阶段：开始/跳过 → 三层测试（10→5→5） → 结果卡片（熟悉/了解/待学）

### 7. 记忆系统

独立面板：自动管理总开关 + 全局性记忆（MD上传/自定义指令）+ 项目记忆（抽象表述/用户画像/自定义），每栏独立滑动开关

### 8. 知识库

独立面板：输入内容（文本输入+文件上传）+ 内容展示（总体概述/知识图谱）+ 状态栏（更新时间/文档数/进度）

### 9. 设置

字体大小滑块 + 日间/夜间/跟随系统 + API Key 配置（保存到浏览器本地）

### 10. 主题

Claude 风格浅色 `#faf8f5` + 深色 `#0d0d0d`
