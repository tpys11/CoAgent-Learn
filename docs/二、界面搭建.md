

## 技术栈

React 19 + TypeScript + Vite 6 + Tailwind CSS 3 + @xyflow/react 12

## 功能总览

### 1. 三栏布局

四周 8px 呼吸间距，三栏 8px 间隙。左侧 240px / 中间弹性 / 右侧 260px。

### 2. 左侧栏 — 项目 + Agent

**上半部分：项目与对话管理（仿 workbuddy）**

* 顶部品牌区（Bot 图标 + 名称 + 新建按钮）

* 每个项目可展开/折叠，显示下属对话窗口列表

* 对话窗口支持：新建（点击项目旁 + 号）、重命名（编辑图标）、归档

* 新建项目后自动创建首个对话窗口并弹出知识诊断

* 项目项橙色高亮，对话项独立选中

**下半部分：Agent 列表**

* 展示系统预设的 Agent（学情诊断 / 知识生成 / 审核裁判）

* 点击 Agent 弹出配置模态框

### 3. Agent 配置模态框

* **模式选择**：每个 Agent 有预设模式（标准 / 深思 / 快速 / 严格），点击切换自动叠加提示词

* **全局性提示词**：可编辑的 System Prompt，带恢复默认按钮

* **Skill**：展示 Agent 的能力描述，部分可编辑（学情诊断的 Skill 系统锁定）

* 所有修改点保存后生效

### 4. 中间 — 对话区

* 反馈横条：黑色边框，显示专注时长 + Token 用量 + 实时时钟

* 消息气泡：用户浅橙右对齐，助手透明左对齐

* 发送中显示脉冲圆点动画

* 控制栏（白底黑边框）：检索模式循环按钮 + 输出形式弹出面板 + 输出内容弹出面板

* 输入框占满宽度，Enter 发送，Shift+Enter 换行

* 消息通过 POST /api/chat 调用后端

### 5. 右侧面板

* 知识图谱占位（虚线边框卡片）

* 追问建议：6 个追问模板按钮

### 6. 知识诊断弹窗

* 三阶段：开始/跳过 → 三层测试（10→5→5） → 结果卡片

* 每题四选项，自定义输入框

### 7. 主题

Claude 风格浅色 `#faf8f5` + 深色 `#0d0d0d`
