# 三、界面搭建 — React 前端

本文档描述 CoAgent-Learn React 前端界面的全部功能。

## 技术栈

| 层次 | 技术 |
|------|------|
| 前端框架 | React 19 + TypeScript |
| 构建工具 | Vite 6 |
| 样式系统 | Tailwind CSS 3 |
| 节点图可视化 | @xyflow/react 12（React Flow，预留） |
| 后端 API | FastAPI 纯 API（端口 8000） |

## 功能总览

### 1. 三栏布局（呼吸感）

- 全屏容器四周留 8px 空白，三栏之间 8px 间隙
- **左侧 240px**：品牌区 + 项目列表 + 新建/删除
- **中间弹性区**：反馈横条 → 消息列表 → 欢迎页 → 输入区
- **右侧 260px**：知识图谱缩略图 + 追问建议

### 2. 左侧项目栏

- 顶部 Bot 图标 + "CoAgent-Learn" 品牌区
- "+ 新建项目" Claude 橙色按钮，弹出命名框
- 新建后自动触发知识诊断弹窗
- 项目选中橙色高亮，悬浮删除按钮

### 3. 知识诊断弹窗

- 新建项目后弹出三阶段弹窗：开始 / 跳过 / 不再提示
- 三层自适应测试：首轮10个宽泛概念 → 第二轮5个收窄 → 第三轮5个深入
- 每题四选项：比较熟悉 / 知道但不熟悉 / 完全不知道 / 自定义输入
- 完成后弹出结果卡片：熟悉 N 项 / 了解 N 项 / 待学 N 项

### 4. 反馈横条

- 黑色边框包裹，显示专注时长 + Token 用量，右上角实时时钟

### 5. 消息区

- 空状态：Bot 图标 + 欢迎文字
- 用户消息：浅橙色背景右对齐
- 助手消息：透明背景左对齐
- 发送中显示三脉冲圆点 + "思考中…"
- 消息通过 `POST /api/chat` 调用后端 API

### 6. 输入区

- 输入框上方控制栏（白底黑边框包裹）：
  - **检索模式**：循环按钮切换 默认 / 增强 / 私有，当前值橙色高亮
  - **输出形式**：弹出面板，包含语言组织（自由/结构化）和输出格式（MD/纯文本）
  - **输出内容**：弹出面板，包含思考展示（关/开）、输出量（精简/适中/拓展）、学习深度（浅/中/深）
  - 面板互斥（同时只打开一个），点击外部关闭
- 输入框占满宽度，发送按钮 Claude 橙色，Enter 发送 Shift+Enter 换行

### 7. 右侧面板

- 知识图谱预览（虚线边框占位卡片，90px 高）
- 追问建议：6 个追问模板按钮

### 8. 主题

- Claude 风格浅色（微黄 `#faf8f5`）+ 深色（黑灰 `#0d0d0d`）
