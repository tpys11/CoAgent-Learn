## 二、功能总览

### 1. 三栏布局（呼吸感）

- **全屏容器** 四周留 8px 空白，三栏之间 8px 间隙
- **左侧** 240px：项目列表 + 新建/删除 + 左下角设置
- **中间** 弹性占满：反馈横条 → 工作过程可视化 → 消息列表 → 欢迎页 → 输入区
- **右侧** 260px：知识图谱缩略图 + 第二对话窗口
- 反馈横条和输入区横向延伸至接近左右栏，两端留 4px 间隙

### 2. 左侧项目栏

- 列表顶部 "+ 新建项目" 按钮，弹出命名对话框
- 新建后弹出"开始知识诊断？"三选一弹窗（开始 / 跳过 / 不再提示）
- 项目项选中高亮，右键菜单可删除
- 左下角齿轮按钮 → 设置面板（浅色/深色主题切换）

### 3. 知识诊断（三层自适应测试）

- 首轮 10 个宽泛概念，根据回答动态生成第二轮 5 个收窄概念，第三轮 5 个深入概念
- 支持退出、跳过、"不再提示"
- 完成后弹出结果卡片：熟悉程度、分类进度条、盲区标签、总结报告

### 4. 反馈横条 + 实时计时器

- **黑色可见边框**包裹，可收起/展开
- 时间段下拉（总时长/30天/7天/1天/本次会话）
- 实时秒级计时，30 秒无操作自动暂停，恢复后自动继续
- 显示专注时长 + Token 用量

### 5. 智能体工作过程可视化

- 4 个步骤：分析输入 → 检索知识库 → 多智能体协同 → 生成输出
- 圆点动画：活跃橙色脉冲，完成后变绿

### 6. 消息区 + 输入区

- 消息气泡渲染（用户消息浅橙背景右对齐，助手消息透明左对齐），Markdown 渲染
- 输入框上方 **带可见边框** 的下拉框行（黑边框包裹）：
  - 检索模式：默认 / 增强 / 私有
  - 输出格式：自由 / 结构化
  - 输出深度：简洁 / 适中 / 深入
  - 思考展示：开 / 关
- Enter 发送，Shift+Enter 换行，发送按钮 Claude 橙色

### 7. 右侧面板

- 知识图谱缩略图（虚线边框占位，90px 高）
- 第二对话窗口：6 个追问模板，多窗口切换

### 8. 主题

- Claude 风格浅色（微黄 `#faf8f5`）+ 深色（黑灰 `#0d0d0d`）
- 系统主题自适应（`dark=None`）
- 所有卡片有可见边框，按钮 hover 有背景变化，下拉框 hover 边框变橙
- 自定义滚动条（5px 宽，圆角）

## 三、文件清单

| 文件 | 职责 |
|------|------|
| `backend/main.py` | FastAPI+NiceGUI入口，三栏全屏布局，进度/诊断/消息/空闲检测接口 |
| `backend/ui/theme.py` | 双主题定义（浅色/深色各13色变量），全局CSS（布局/面板/消息/按钮/输入框/滚动条/动画/步骤圆点/下拉框） |
| `backend/ui/sidebar.py` | 左侧项目栏：新建/选择/删除 |
| `backend/ui/diagnosis.py` | 三层自适应知识诊断弹窗 + 结果富卡片 |
| `backend/ui/stats_bar.py` | 反馈横条：收起/展开，时间段切换，实时计时器（30秒空闲暂停），token计数 |
| `backend/ui/chat_input.py` | 输入框 + 4个检索/输出设定下拉框（带可见边框） |
| `backend/ui/settings.py` | 设置面板：主题切换 |
| `backend/ui/second_window.py` | 第二对话窗口：6追问模板 + 多窗口切换 |
| `run.py` | 启动脚本（设置路径→加载配置→启动uvicorn） |
| `start.bat` | Windows 双击快捷启动 |

## 四、验证方式

浏览器打开 `http://localhost:8000`：

1. 三栏布局占满全屏，背景微黄
2. 点击 "+ 新建项目" → 弹出对话框 → 创建后弹出诊断弹窗
3. 诊断测试四选项可点击，完成后弹出结果卡片
4. 反馈横条可收起/展开，时间段可切换，计时器实时运行
5. 输入框上方 4 个下拉框可切换
6. 右侧栏知识图谱占位 + 追问按钮可见
7. 左下角齿轮 → 主题切换
8. 消息发送后中间区显示步骤圆点动画
