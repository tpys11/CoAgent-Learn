# 三、界面搭建 — React 前端

本文档描述 CoAgent-Learn React 前端界面的全部功能。

## 技术栈

| 层次 | 技术 |
|------|------|
| 前端框架 | React 19 + TypeScript |
| 构建工具 | Vite 6 |
| 样式系统 | Tailwind CSS 3 |
| 节点图可视化 | @xyflow/react 12（React Flow，预留） |
| 后端 API | FastAPI 纯 API（端口 8000） |

## 功能总览

### 1. 三栏布局（呼吸感）

- 全屏容器四周留 8px 空白，三栏之间 8px 间隙
- **左侧 240px**：项目列表 + 新建/删除 + 设置
- **中间弹性区**：反馈横条 → 工作过程可视化 → 消息列表 → 欢迎页 → 输入区
- **右侧 260px**：知识图谱缩略图 + 第二对话窗口

### 2. 左侧项目栏

- "+ 新建项目" 按钮，弹出命名对话框
- 新建后弹出"开始知识诊断？"三选一弹窗
- 项目项选中高亮，右键删除

### 3. 知识诊断

- 三层自适应测试：首轮10个 → 第二轮5个 → 第三轮5个
- 完成后弹出结果卡片：熟悉程度、分类进度条、盲区标签

### 4. 反馈横条 + 计时器

- 黑色边框包裹，显示专注时长 + Token 用量

### 5. 消息区 + 输入区

- 消息气泡（用户浅橙右对齐，助手透明左对齐）
- 输入框上方 4 个下拉框（白底黑边框）：检索模式/输出格式/输出深度/思考展示
- Enter 发送，Shift+Enter 换行

### 6. 右侧面板

- 知识图谱预览（虚线边框占位）
- 第二对话窗口：6 个追问模板

### 7. 主题

- Claude 风格浅色（微黄 `#faf8f5`）+ 深色（黑灰 `#0d0d0d`）

## 启动方式

双击 `D:\desktop\start-coagent.bat`：Docker Compose 启动 4 服务 → 自动打开浏览器 `http://localhost:5173`。
