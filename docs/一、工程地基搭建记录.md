## 三、技术选型对齐

| 层次 | 选型 | 说明 |
|------|------|------|
| 后端框架 | FastAPI | 纯 API，高性能异步 |
| 前端框架 | React 19 + Vite 6 + Tailwind CSS | 独立进程，节点图预留 xyflow |
| 多智能体 | LangGraph | 编排三 Agent：学情诊断→知识生成→审核裁判 |
| 工具调用 | MCP | 标准化协议 |
| 模型接入 | BaseLLM 基类 → DeepSeek | OpenAI SDK 统一封装 |
| 业务数据库 | SQLite | 轻量持久化 |
| 向量数据库 | Chroma | RAG 知识检索 |
| 缓存 | Redis | 任务队列与临时存储 |
| 部署 | Git + Docker Compose | 一键四服务编排 |

---

## 四、Docker Compose 四服务编排

| 服务 | 镜像/构建 | 端口 | 作用 |
|------|----------|:----:|------|
| frontend | node:22-alpine 构建 | 5173 | React 前端 |
| backend | python:3.12-slim 构建 | 8000 | FastAPI API |
| redis | redis:7-alpine | 6379 | 缓存队列 |
| chroma | chromadb/chroma:latest | 8001 | 向量数据库 |

用户只需访问 5173 一个端口，Vite 代理 `/api` → 8000。桌面 `start-coagent.bat` 一键 `docker compose up -d` 启动。

---

## 五、Python 开发环境

`python -m venv .venv` 虚拟环境。`backend/requirements.txt` 含 11 个核心依赖：FastAPI、Uvicorn、LangGraph、LangChain、langchain-deepseek、ChromaDB、Redis、SQLAlchemy、python-dotenv、pydantic、httpx。`backend/main.py` 实现纯 API 骨架，`/health` 存活探测，`POST /api/chat` 对话接口。
