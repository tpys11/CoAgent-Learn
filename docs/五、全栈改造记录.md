# 五、全栈改造 — React 前端 + 后端优化记录

> 日期：2026年7月22日

## 技术栈变更

根据最新技术选型方案，将前端从 NiceGUI（Python 全栈）迁移为 React 独立前端框架：

| 层次 | 技术 |
|------|------|
| 前端框架 | React 19 + TypeScript |
| 构建工具 | Vite 6 |
| 样式系统 | Tailwind CSS 3 |
| 节点图可视化 | @xyflow/react 12（React Flow） |
| 后端框架 | FastAPI |
| 多智能体 | LangGraph + MCP |
| 项目构建 | pyproject.toml（PEP 621） |

## 前端实现的功能

### 1. 三栏布局

- **左侧 240px**：项目列表侧栏，背景独立于主内容区。支持新建项目（输入名称或自动命名）、项目选中高亮切换、右键删除
- **中间弹性区**：消息对话区 + 输入区，占满可用宽度，左右留 8px 呼吸间距
- **右侧 260px**：知识图谱预览区 + 第二对话窗口

### 2. 消息对话系统

- 用户消息：浅橙背景气泡右对齐，带橙色半透明边框
- 助手消息：透明背景左对齐，hover 有浅灰底色
- 欢迎页：空状态时显示 Bot 图标 + 应用名 + 当前项目提示

### 3. 输入区域

- 4 个下拉框设定：检索模式（默认/增强/私有）、输出格式（自由/结构化）、输出深度（简洁/适中/深入）、思考展示（开/关）
- 所有下拉框白底黑边框包裹，与背景清晰区分
- 输入框占满宽度（延伸至接近左右栏），发送按钮 Claude 橙色
- Enter 发送，Shift+Enter 换行

### 4. 反馈横条

- 黑色边框包裹，显示专注时长 + Token 用量 + 实时计时器
- 位于输入区上方，左右延伸至接近两栏边缘

### 5. 右侧面板

- 知识图谱占位卡片（虚线边框）
- 第二对话窗口：6 个追问模板（相关概念/应用场景/深入学习/知识连接/实践练习/检验理解）

### 6. 后端优化

- pyproject.toml 声明 12 个核心依赖（移除 nicegui/redis，新增 mcp≥1.0）
- 输入框样式优化：下拉框白底区分 + 左右延伸
- 反馈横条黑边框
- 移除三模式选择，简化为统一默认模式
- 四周 8px 呼吸间距

## 启动方式

| 服务 | 命令 | 端口 |
|------|------|------|
| 后端 | `python run.py` 或双击 `start-backend.bat` | 8000 |
| 前端 | `cd frontend && npm run dev` 或双击 `start-frontend.bat` | 5173 |
