# 零、项目启动方式

## 原理

本项目后端使用 **FastAPI + NiceGUI** 构建。关键点：

- `backend/main.py` 创建 FastAPI 应用，通过 `ui.run_with(app)` 将 NiceGUI 挂载到 FastAPI 上
- **`ui.run_with()` 仅配置路由，不启动 HTTP 服务器**
- 必须额外调用 `uvicorn.run(app, host='127.0.0.1', port=8000)` 才能真正启动

## 启动方式

### 方式一：python run.py（推荐）

```powershell
cd D:\desktop\guashuai-project
.venv\Scripts\python run.py
```

`run.py` 自动完成：添加项目根目录到 `sys.path` → 加载 `.env` 环境变量 → 从 `backend.main` 导入 `app` → 启动 uvicorn。

### 方式二：双击 start.bat

桌面 `start.bat` 或项目根目录 `start.bat`，内容等价于方式一。

### 方式三：原始方式

```powershell
cd D:\desktop\guashuai-project
.venv\Scripts\python -c "import sys; sys.path.insert(0, '.'); from backend.main import app; import uvicorn; uvicorn.run(app, host='127.0.0.1', port=8000)"
```

## 注意事项

- 需要 Python 虚拟环境 `.venv`（已配置好）
- 如需 Agent 功能，提前在 `.env` 中配置 `DEEPSEEK_API_KEY`
- 无 API Key 时进入界面预览模式，UI 正常可用
- 服务启动后访问 `http://localhost:8000`
- 按 `Ctrl+C` 停止
