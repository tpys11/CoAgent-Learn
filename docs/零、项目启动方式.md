## 原理

CoAgent-Learn 由 4 个 Docker 容器组成：

| 服务       |  端口  | 作用                               |
| -------- | :--: | -------------------------------- |
| frontend | 5173 | React 19 前端，用户界面                 |
| backend  | 8000 | FastAPI 纯 API，LangGraph Agent 编排 |
| redis    | 6379 | 缓存与任务队列                          |
| chroma   | 8001 | 向量数据库，RAG 知识检索                   |

前端 Vite 代理 `/api` → 后端 8000。用户只需访问 5173 一个端口。

## 启动方式

### 方式一：双击桌面脚本

桌面文件：`start-coagent.bat`

```bat
@echo off
docker compose -f "D:\desktop\coAgent-Learn\deploy\docker-compose.yml" up -d
timeout /t 4 /nobreak >nul
start http://localhost:5173
```

自动启动 Docker → 等 4 秒 → 自动打开浏览器

### 方式二：前后端分开（适用于本地开发）

```powershell
# 终端1 - 后端
cd D:\desktop\coAgent-Learn
.venv\Scripts\python run.py

# 终端2 - 前端
cd D:\desktop\coAgent-Learn\frontend
npm run dev
```

## 前置条件

* Docker Desktop 已安装并运行

* `.env` 配置 `DEEPSEEK_API_KEY`（否则 Agent 不可用，UI 正常）

* 首次使用前端需 `cd frontend && npm install`
