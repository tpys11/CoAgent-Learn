# 零、项目启动方式

## 原理

CoAgent-Learn 由 4 个 Docker 容器组成，`docker compose up -d` 一键全启：

| 服务 | 端口 | 作用 |
|------|:----:|------|
| frontend | 5173 | React 19 前端（Vite 代理 /api → 8000） |
| backend | 8000 | FastAPI + LangGraph 多智能体协同 |
| redis | 6379 | 缓存与任务队列 |
| chroma | 8001 | 向量数据库（RAG 知识检索） |

用户只需访问 5173 一个端口。

## 首次配置

编辑 `D:\desktop\coAgent-Learn\.env`，填入 DeepSeek API Key：

```
DEEPSEEK_API_KEY=sk-你的真实key
```

## 启动方式

### 方式一：双击桌面脚本（推荐）

双击 `D:\desktop\start-coagent.bat`：

```bat
docker compose -f "D:\desktop\coAgent-Learn\deploy\docker-compose.yml" up -d
start http://localhost:5173
```

自动启动 Docker 服务 → 打开浏览器。

### 方式二：命令行

```powershell
docker compose -f "D:\desktop\coAgent-Learn\deploy\docker-compose.yml" up -d
```

浏览器打开 `http://localhost:5173`。

**停止**：`docker compose -f "D:\desktop\coAgent-Learn\deploy\docker-compose.yml" down`

## 前置条件

- Docker Desktop 已安装并运行
- `.env` 中已配置 `DEEPSEEK_API_KEY`
- 无需安装 Node.js 或 Python（全部在容器内）
