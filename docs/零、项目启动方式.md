# 零、项目启动方式

## 原理

- **后端**：FastAPI + NiceGUI，`ui.run_with(app)` 配置路由，`uvicorn.run()` 启动 HTTP 服务器
- **前端**：React 19 + Vite，独立进程运行，通过 `vite.config.ts` 代理 `/api` 请求到后端 8000 端口

## 启动方式

### 一键启动（推荐）

双击 `start-all.bat`，后端和前端各开一个窗口同时启动：
- 后端 `http://localhost:8000`
- 前端 `http://localhost:5173`

### 分步启动

**后端**：
```powershell
cd D:\desktop\guashuai-project
.venv\Scripts\python run.py
```
或双击 `start-backend.bat`

**前端**：
```powershell
cd D:\desktop\guashuai-project\frontend
npm run dev
```
或双击 `start-frontend.bat`

## 注意事项

- Python 虚拟环境 `.venv` 已配置
- Agent 功能需 `.env` 中配置 `DEEPSEEK_API_KEY`，无 Key 时 UI 正常可用
- 前端首次启动需 `cd frontend && npm install`（已执行过则跳过）
- 访问前端 `http://localhost:5173` 即可使用全部功能
